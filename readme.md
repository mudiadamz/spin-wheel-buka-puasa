## Versi Next.js

Ada implementasi Next.js di folder `nextjs/`. Jalankan dengan:
```bash
cd nextjs && npm run dev
```

---

- idenya adalah:
- spin wheel berisi makanan untuk berbuka puasa
- terdiri dari main course, side dish, dessert, minuman

- flow:
- speen will berwarna warni
- user menekan tombol spin ditengah wheel
- tampilan spin wheel 3d
- ketika ditekan animasi berputar
- berputar seperti asli, melambat saat mau selesai
- ketika selessai baru muncul tampilan makanan

tampilan:
- mobile dengan max width
- warna menarik khas ramadan
- ada sidebar menu dengan isi: history spin, about, contact, privacy policy, 30 hari jadawal berbuka
- ada button untuk merefresh jenis makanan
- ada history disimpan di local storage
- login dengan google untuk menampilkan history spin dan notifikasi firebase
